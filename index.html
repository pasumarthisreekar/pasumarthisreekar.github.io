Structure Preserving Deep Autoencoder based Dimensionality Reduction for Data Visualization
thesis description

Autoencoders are typically used for the task of dimensionality reduction. However, a drawback exists. If an autoencoder has additional representative capacity, it may not preserve the structure of data faithfully in the lower-dimensional space. Regularisation can be a way to solve this problem, as it can constrain the autoencoder from merely copying the input to the output. Here, we propose a regulariser that aims to preserve the geometric properties of the input data by preserving the geodesic distances between data points in the input space as the Euclidean distance in the embedding space. Further, for the purpose of data visualization, we set the lower dimensionality to 2 or 3. We then test the visual quality of the embeddings and observe that the autoencoder was able to preserve the manifold structure successfully.
